# Introduction

## Installation

```bash
pip install aitermis
```

## Upgrading to the latest version

```bash
pip install aitermis -U
```

## Basic usage example

### Input

```bash
a "create new next.js app"
```

### Output

```bash
your query: create new next.js app
command: npx create-next-app@latest myapp
```
